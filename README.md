##### Family Tree
###### 简介
- 用`c++`实现的命令行操作界面的家谱

###### 版本介绍
- `code`目录下对应不同的目录有不同的功能
  - original：第一个版本，只有普通操作
  - operation-record：记录每一次输入操作
  - all-record：记录输入输出全过程
  - auto：直接从文件读取并执行操作，操作全过程将会被记录

###### 执行
- 进入到目录，执行`g++ main.cpp`即可操作

###### Test-file
- 用于测试家谱的操作
- 可把它复制到`auto`目录并替代`input`即可实现测试
